This directory contains the raw NASA C-MAPSS FD001 dataset files.

Required files:
  - train_FD001.txt   (~4.1 MB, 20631 rows × 26 columns)
  - test_FD001.txt    (~1.6 MB, 13096 rows × 26 columns)
  - RUL_FD001.txt     (~0.5 KB, 100 values)

Download from:
  - NASA: https://www.nasa.gov/content/prognostics-center-of-excellence-data-set-repository
  - Kaggle: https://www.kaggle.com/datasets/behrad3d/nasa-cmaps

Total dataset size: ~6 MB (small enough to track in Git directly).
